# Supermarket Admin — Products Manager

## Project Overview
This is a full-stack Supermarket Application built for Web Development Assignment 02. It features a complete RESTful API backend and a responsive, modern frontend UI designed for managing supermarket inventory. It supports full CRUD operations, including multipart form data for image uploads.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript (Fetch API)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Middleware/Tools:** Multer (File Uploads), CORS, Dotenv
